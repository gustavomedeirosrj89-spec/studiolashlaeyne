# **App Name**: LAEYNE STUDIO LASH

## Core Features:

- AI Look Matcher Tool: An AI-powered tool that analyzes user eye shape descriptions and style preferences to suggest the perfect lash volume style.
- Service Discovery Portfolio: An interactive showcase of the studio's specialized models like Volume Brasileiro and Fox Eyes with detailed descriptions and pricing.
- Integrated Booking Connector: One-touch action buttons to instantly transition potential clients to WhatsApp for appointment confirmation and consulting.
- Dynamic Review Showcase: A scroll-revealed gallery of verified client feedback to build studio authority and social proof.
- Full Service Digital Catalog: A comprehensive overlay containing all maintenance intervals, removal fees, and advanced treatment tiers.
- Payment & Logistics Module: Quick-reference cards highlighting Pix, credit, and cash payment methods for maximum customer convenience.

## Style Guidelines:

- Primary color: Muted Satin Gold (#A68559) to evoke luxury and delicate craftsmanship.
- Background: Warm Bone Nude (#F7F3EE) chosen for a sophisticated light color scheme that provides soft contrast.
- Accent: Dusty Rose Terracotta (#B68D84) used for interactive states and decorative highlights.
- Headlines and logo: 'Cormorant Garamond' (serif) for a high-fashion, vintage feel. Body: 'Jost' (sans-serif) for modern clarity and contemporary elegance.
- Minimalist, thin-stroke SVG line icons featuring sparks, lashes, and organic geometric shapes.
- A high-end editorial layout using asymmetric grids, wide white space, and blob-styled imagery frames.
- Soft cubic-bezier entrance animations that reveal elements gradually as the user scrolls, creating a 'graceful appearance' effect.
