# LAEYNE Studio Lash - Gestão de Luxo

Sistema de gestão premium para lash designers, focado em experiência do cliente e automação de agendamentos.

## 🚀 Como começar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## 🛠️ Tecnologias Utilizadas

- **Next.js 15 (App Router)**
- **Firebase** (Auth, Firestore)
- **Genkit AI** (Sugestões personalizadas de estilos)
- **Tailwind CSS** & **ShadCN UI**
- **Date-fns** (Gestão inteligente de horários)

## 📦 Configuração do Repositório (Git)

Para subir o projeto para o seu repositório GitHub pela primeira vez:

```bash
git init
git add .
git commit -m "Primeiro commit do projeto Studio Lash Laeyne"
git remote add origin https://github.com/gustavomedeirosrj89-spec/studiolashlaeyne.git
git branch -M main
git push -u origin main
```

## 💎 Características do Projeto

- **Arquitetura de Luxo**: Design minimalista e editorial.
- **Agendamento Inteligente**: Verificação de conflitos em tempo real e slots dinâmicos de 30 minutos.
- **Painel Administrativo**: Gestão completa de clientes, serviços e agenda.
- **IA Integrada**: Look Matcher para sugestão de volume baseado no formato dos olhos.
